## 프로젝트 소개

실시간 입찰 흐름을 가진 경매 서비스입니다.
백엔드 API와 연동하여 상품 조회, 입찰, 경매 상태 관리, 입찰 내역 갱신 기능을 구현했습니다.

## 핵심 구현

- TanStack Query 기반 상품/입찰 API 캐싱 및 재요청 제어
- 경매 상태에 따른 UI 분기 처리
- 입찰 금액 유효성 검증
- 경매 종료 시간 기준 카운트다운 구현
- 로딩/에러/빈 상태 UI 처리
- 반응형 레이아웃 적용

## 고민한 점

- 입찰 후 화면 갱신을 어떻게 처리할지
- 경매 종료 상태를 프론트에서 어떻게 안전하게 표시할지
- API 실패 시 사용자 경험을 어떻게 처리할지
